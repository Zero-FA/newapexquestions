Common Questions and Troubleshooting for WealthCharts

Overview
This guide answers common questions about the WealthCharts platform, including trading features, system requirements, alerts, and troubleshooting.


Notifications and Alerts

Can WealthCharts notify me when price reaches my parameters?
Yes. WealthCharts can notify you through:

- Email
- SMS
- In-app notifications

Alerts can be triggered when:

- A scanner condition is met
- A specific price level is reached

Scanner alerts and price alerts allow traders to step away from the screen while still being notified when trading conditions occur.


Contracts and Futures Selection

What contract should I trade?
Front-month futures contracts are determined by the highest open interest from the prior trading session.

To identify the front-month contract:

1. Visit https://www.cmegroup.com/markets/products.html#sortAsc
2. Select the desired futures product
3. Click the “Volume & OI” tab
4. Ensure “Futures Quote” data is selected (not Options Quote)
5. Check the Open Interest column

The contract month with the highest open interest is considered the front-month contract.


Indicators and Custom Tools

Can I create custom indicators in WealthCharts?
Yes. WealthCharts includes a scripting language called WealthScript that allows users to build custom indicators and analysis tools.

If you do not write code, you can use the WealthScript Visual Builder to create custom tools using a drag-and-drop interface.


Trading on WealthCharts

Can I trade directly on WealthCharts?
Yes. WealthCharts supports direct trading through multiple interfaces including:

- Chart trading
- Order panels
- DOM (Depth of Market) trading
- Options trading tools

Traders can also place custom stop orders and manage trades directly from charts.


Platform Installation

Do I need to download WealthCharts?
No. WealthCharts is a web-based platform.

It can be accessed through a web browser from any computer.

If you are away from your main computer, you can:

- Log in from another computer through a browser
- Use the WealthCharts mobile app available on Apple App Store and Google Play


Operating System Compatibility

Does WealthCharts work on Windows and Mac?
Yes. Because WealthCharts is web-based, it works on both Windows and Mac computers.

For best performance, make sure your browser is updated to the latest version.


Paper Trading

Does WealthCharts offer paper trading?
Yes. WealthCharts provides its own paper trading system.

This allows traders to simulate trades in real-time without connecting to a broker.

WealthCharts also offers additional practice tools including:

- Trading Simulator
- Chart Replay for historical market data

These tools help traders practice strategies before trading live.


System Requirements

What are the computer requirements for WealthCharts?
WealthCharts runs inside a web browser.

For best performance:

- Use the latest version of your browser
- Recommended browsers include Google Chrome and Microsoft Edge


WealthCharts Charting Features

WealthCharts provides advanced charting tools including:

These tools allow traders to customize their analysis environment.

Platform Compatibility

Can I convert a WealthCharts evaluation into a Tradovate or Rithmic Performance Account?
No. WealthCharts evaluations can only be converted into WealthCharts Performance Accounts.

If you want a Tradovate or Rithmic Performance Account, you must pass the evaluation for that specific platform.


Account Login Issues

What if I forgot my WealthCharts password?
Visit the login page:

https://app.wealthcharts.com/login

Use the “Request One-Time Password” option.

A one-time password will be sent to your email which allows you to log in and set a new password.


Trade Copier Help

I am having issues with the Trade Copier
Log in to the WealthCharts dashboard and locate the Trade Copier Panel.

Inside the panel there is a tutorial icon which opens a video explaining how the Trade Copier works.

Additional questions about the Trade Copier can be asked in the WealthCharts Live Training room.


Password Reset Email Not Received

If you request a password reset but do not receive an email, it may be because your email address was unsubscribed from their mailing list.

To resolve this, contact WealthCharts support directly:

Email: support@wealthcharts.com  
Subject line: Password Reset
